Negrito e Itálico

Texto escrito em negrito com a tag <b> é errado. O correto é utilizar a tag <strong>

Para itálico também existe a tag errada <i>. A tag correta é <em>

Texto Marcado

Podemos marcar um texto usando a tag <mark>

Texto Grande e Pequeno

A tag <big> para deixar as letras grandes se tornou obsoleta, ja <small> não

Texto Inserido

Para sublinhar pode-se utilizar a tag <u>, porém a correta é a <ins>.

Texo Sobrescrito e Subescrito

Para sobbrescrever um texto utilize a tag <sup>. Ex: x3

Ja para subescrever utilize a tag <sub>.Ex: H2O
